Frontend Quiz App Site Plan

Site Name

Frontend Quiz

An interactive quiz application designed to test and improve knowledge of frontend development technologies. The app provides multiple quiz categories including HTML, CSS, JavaScript, and Accessibility, helping developers assess their skills and learn new concepts through engaging quiz questions.

Site Purpose

The Frontend Quiz App is designed to help developers test and enhance their frontend knowledge. The application provides:

  • Multiple quiz categories covering essential frontend technologies
  • Interactive question and answer format with immediate feedback
  • Score tracking and progress monitoring
  • Light and dark theme options for better user experience
  • Keyboard navigation support for accessibility
  • Responsive design optimized for mobile, tablet, and desktop

Scenarios

  • How can I test my HTML knowledge and identify areas for improvement?
  • What CSS concepts should I focus on to advance my skills?
  • How well do I understand JavaScript fundamentals and modern practices?
  • Am I following accessibility best practices in my web development?
  • How can I track my progress across different frontend topics?
  • Can I practice frontend concepts on my mobile device during commute?

Typography

Primary Font: Rubik

Usage:

  • Headings (Medium - 500, Semibold - 600)
  • Body Text (Regular - 400)
  • Questions and Answers (Regular - 400, Medium - 500)
  • Buttons and Interactive Elements (Medium - 500)
  • Navigation and Labels (Regular - 400)

Rubik provides excellent readability and modern feel perfect for a quiz application interface.

Wireframes

Mobile View

Mobile interface will feature:

  • Full-screen quiz interface with clear question display
  • Touch-friendly answer buttons with proper spacing
  • Progress indicator showing current question position
  • Theme toggle accessible from header
  • Simplified navigation optimized for touch interaction
  • Responsive typography scaling for readability

Desktop View

Desktop layout will utilize:

  • Two-column layout with question on left, answers on right
  • Prominent theme toggle in the header
  • Hover states for interactive elements
  • Keyboard navigation support for accessibility
  • Larger typography and spacing for comfortable reading
  • Progress bar and scoring clearly displayed
